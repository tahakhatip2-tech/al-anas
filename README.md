# الأُنس | Al-Ons Event Photography & Management

مشروع احترافي متخصص في إدارة الفعاليات وتصوير المناسبات، مصمم لتقديم تجربة مستخدم استثنائية وعرض الخدمات الفاخرة التي يقدمها فريق الأُنس.

## التقنيات المستخدمة (Tech Stack)

تم بناء هذا المشروع باستخدام أحدث تقنيات الويب لضمان السرعة والكفاءة:

- **React**: مكتبة الواجهة الأمامية.
- **Vite**: أداة بناء سريعة جداً.
- **TypeScript**: لضمان استقرار الكود وتقليل الأخطاء.
- **Tailwind CSS**: للتصميم العصري والمرن.
- **Shadcn UI**: مكونات جاهزة واحترافية.
- **Framer Motion**: للتحركات والمؤثرات البصرية.
- **TanStack Query (React Query)**: لإدارة البيانات.

## الميزات الرئيسية

- **دعم كامل للغتين (عربي/إنجليزي)** مع تبديل سلس.
- **دعم اتجاه النصوص (RTL/LTR)**.
- **وضع ليلي (Dark Mode)** ووضع نهاري متناسق.
- **تصميم متجاوب** يعمل على جميع الأجهزة.
- **تكامل مع واتساب** للحجز المباشر.
- **معرض أعمال** جذاب ومتحرك.

## التشغيل محلياً

اتبع الخطوات التالية لتشغيل المشروع على جهازك:

```bash
# 1. تثبيت الحزم والمكتبات
npm install

# 2. تشغيل سيرفر التطوير
npm run dev
```
